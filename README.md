# UEx Screensaver
El clásico salvapantallas del logo de DVD(TM) vuelve de nuevo, ahora reimaginado y tematizado para la UEx. 

Binarios disponibles para Windows y Linux en la sección de [Releases](https://github.com/piporgames/unexCorner/releases).   

## Funcionamiento
El salvapantallas se conecta al campus universitario y recupera una imagen aleatoria de algún usuario de la plataforma cada vez que la imagen choca con el borde del monitor.

## Información

- `--step <valor>`: Define velocidad de movimiento. Por defecto: `5`.
- `--queue <valor>`: Establece el tamaño del buffer de imagenes. Por defecto:`5`.
- `--min <valor>`: Especifica el ID de usuario mínimo para la generación aleatoria. Por defecto: `2278300`.
- `--max <valor>`: Especifica el ID de usuario máximo para la generación aleatoria. Por defecto: `2278600`.
  
Escrito en Python 3.
